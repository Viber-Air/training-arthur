API

A api utiliza o método QuerySet.filter() do Django. Assim, a maior parte das palavras-chave são compatíveis. Para mais informações: https://docs.djangoproject.com/en/4.0/ref/models/querysets/#field-lookups-1

Paravras-chave úteis:

  • gt - greater than
  • gte - greater than or equal to
  • lt - less than
  • lte - less than or equal to
  • range(   ,   ) - range test (inclusive)

Exemplos

/api/[Model/Collection]/?filters

* para o exemplo usaremos model=rawdata

  • Mostrando todos os valores:

http://localhost:8000/api/rawdata

  • Somente dados do sensor com sensor_id=1:

http://localhost:8000/api/rawdata/?sensor_id=1

  • Somente apartir do dia 01/04/22:

http://localhost:8000/api/rawdata/?initial_time__gt=2022-04-01

  • Somente apartir do dia 01/04/22 às 12h30m00s:

http://localhost:8000/api/rawdata/?initial_time__gt=2022-04-01T12:30:00

  • Somente entre os dias 01/04/22 e 01/05/22:

http://localhost:8000/api/rawdata/?initial_time__range=(2022-04-01,2022-05-01)
